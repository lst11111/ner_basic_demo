# ner_basic_demo

## 项目描述

该仓库旨在利用 BERT 模型解决命名实体识别（NER）任务。我们采用了微博（Weibo）数据集进行训练和评估，目标是提升中文文本中的实体识别能力。经过训练，模型在开发集（dev）上取得了约 69 的 F1 值，在测试集（test）上达到了约 64 的 F1 值。该项目展示了基于预训练语言模型（如 BERT）在中文 NER 任务中的应用，并为相关研究提供了一个基础框架。

## 主要特点

- 使用 BERT 进行命名实体识别任务。
- 基于微博数据集进行模型训练和评估。
- 在开发集上实现了约 69 的 F1 分数。
- 在测试集上实现了约 64 的 F1 分数。
